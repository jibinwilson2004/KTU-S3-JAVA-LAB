# KTU S3 OOPS Java Lab Programs

This page mirrors the repository README to help search engines index the content.

## Overview
18 Java programs for KTU OOPS: palindrome, string utilities, matrix multiplication, inheritance, abstract classes & polymorphism, garbage collector demo, file handling (Reader/Writer, streams, exceptions), StringTokenizer, exception flow, multithreading, synchronization, Swing (calculator, traffic light), JDBC table display, doubly linked list, quick sort (names), and binary search.

## Quick Start
```bash
javac Palindrome.java && java Palindrome
javac SwingCalculator.java && java SwingCalculator
javac JDBCDisplayAllRecords.java && java JDBCDisplayAllRecords "jdbc:your:URL" user pass table
```
